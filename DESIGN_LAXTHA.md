## 3. 페이지 구조

### 홈 (index.html)
1. 유틸바 + 헤더(검색) + (모바일 카테고리 토글)
2. 본문 = [좌 사이드바] + [메인]
   - **히어로**: 실제 사이트의 "3D 뇌 + 생체신호 파형 + 생체신호 전문 쇼핑몰" 컨셉을 현대적으로 — 밝은 카드형 히어로, 좌측 카피("생체신호 측정의 시작, LAXTHA" / "ECG·EEG·EMG 센서·전극·소프트웨어") + 듀얼 CTA(제품 보기/견적 문의) + 우측 **차분한 생체신호 파형 SVG**(브랜드 모티프). 슬라이더 점은 정적 1~2개 표현 가능.
   - **카테고리 하이라이트(3)**: 뇌파(EEG)/맥파·심전도(ECG)/근전도(EMG) — 실제 대표 제품 사진 카드, 클릭 시 해당 카테고리.
   - **NEW PRODUCT / 추천 제품 그리드**: 실제 19종. 정렬/필터 미니탭(전체·EEG·EMG·ECG). 제품 카드(아래 컴포넌트).
   - 신뢰 스트립(연구기관 도입·KFDA·기술지원) 작게.
3. 푸터: 회사정보 + 카테고리/브랜드/고객센터 링크 + 사업자정보 + 에스크로/약관.

### 카테고리/검색 (products.html?cat= / ?q=)
- 빵부스러기(홈 > 카테고리명) + 카테고리 타이틀 + "전체 상품 N개"
- 정렬바: 인기순·최근등록순·낮은가격순·높은가격순 + 뷰토글(그리드/리스트) — 실제 사이트 그대로 현대화
- 좌 사이드바 유지 + 메인 제품 그리드(3~4열, 모바일 2열). 검색·브랜드 필터 동작.

### 상세 (product.html?id=)
- 빵부스러기 + [좌 큰 제품 이미지] + [우 정보: 카테고리/브랜드 칩, 제품명, 모델, 사양 표, 가격(큰 숫자), 수량 스텝퍼, 장바구니/바로구매, 배송·견적 안내] + 하단 동일 카테고리 추천.

### 장바구니/체크아웃/주문완료
- 기존 검증된 흐름·결제(KCP/PortOne 테스트) 그대로, 새 디자인으로 재구성.

## 4. 컴포넌트
- **제품 카드**: 흰 배경, 1px `--c-hairline`, `--r-lg`, hover `--shadow-md`+틸 보더. 상단 정사각 이미지(흰 패딩), 하단: 브랜드·카테고리 작은 라벨(신호색 도트), 제품명 2줄, 사양 회색, **가격 `--font-num` 잉크/틸**, [장바구니] 틸 버튼(아이콘+텍스트). 카드 전체 클릭 → 상세.
- **버튼**: primary=틸(흰 텍스트, `--r-md`), secondary=흰 배경+틸 보더, ghost=텍스트 틸.
- **검색바**: 큼직, `--r-pill` 또는 `--r-md`, 좌 아이콘, 우 틸 검색버튼.
- **사이드바 카테고리**: 리스트, 현재 카테고리 틸 강조 바, hover 연틸 배경.
- **칩/뱃지**: 카테고리=신호색 연한 배경, "NEW/추천"=틸/네이비.
- **퀵메뉴 레일**: 흰 카드 세로 아이콘 버튼들 + 카운트.

## 5. 비주얼 규칙
- 토큰: `public/assets/css/tokens.css`(Clinical Light) var()만. 하드코딩 금지.
- 흰/연그레이(`--c-canvas-alt`) 교차. 카드 소프트섀도. 라운드 8~16px. 여백 넉넉.
- 옐로/검정 거래소 톤 0. 틸은 액션/강조에만. 신호색은 카테고리 구분·파형에만.
- 타이포: 제목 Inter 700 `--c-ink`, 본문 `--c-body`, 숫자 IBM Plex Sans.
- 접근성: 대비 AA, alt, aria, `:focus-visible` 틸 ring. 반응형: 데스크톱(좌 사이드바+퀵레일) / 태블릿(사이드바 접힘) / 모바일(헤더 햄버거→카테고리 드로어, 그리드 2열, 퀵레일 숨김).

## 6. 재사용(변경 금지) & 신규
